# Real Estate Valuation Model - BPRE

A compact investment-banking style valuation model for a Building, Property & Real Estate (BPRE) coverage role.

## What it includes

- Operating forecast
- DCF valuation
- Trading comparables valuation
- NAV / cap-rate valuation
- EV/EBITDA sensitivity analysis
- Cap-rate / NOI sensitivity analysis
- Summary page with valuation range and CV-ready portfolio wording

## Why this project is relevant

This model demonstrates core investment banking analyst skills:

- Financial modelling
- Valuation
- Company and sector analysis
- Trading comparables
- DCF methodology
- Real estate / cap-rate valuation
- Sensitivity analysis
- Pitchbook-style summary output

## Important note

This model uses illustrative placeholder assumptions. It is for portfolio and learning purposes only and is not investment advice. Replace all assumptions with live filings, market data, broker research or company guidance before using it professionally.
